Some commonly used vim shortcuts

## Sudo TEE trick

```bash
:w !sudo tee %
```

## Duplicate Line
```bash
# In command mode
Vyp

# V - Select Line
# y - Copy Line
# p - Paste Line After
# P - Paste line before
```

## Look for autocomplete
```bash
# Insert Mode
CTRL + p
```
