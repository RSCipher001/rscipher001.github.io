This page lists common commands in firebase cli

## Installation
```bash
npm i -g firebase-tools
# i  - Install
# -g - Global
```

## Login
This will open a browser where you can login your Google account
```bash
firebase login
```

## List project
```bash
firebase projects:list
```

## Start a project
```
firebase init
```
